RSS-Bridge

The RSS feed for websites missing it

View On GitHub View Documentation

RSS-Bridge is free and open source software for generating Atom or RSS feeds from websites which don’t have one. It is written in PHP and intended to run on a Web server. See the Screenshots for a quick introduction to RSS-Bridge
